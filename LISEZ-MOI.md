# 🎮 BLURRED — Comment jouer

**BLURRED** est un jeu d'enquête (escape game) où vous devez démasquer le coupable
de la mort de Louis en fouillant l'appartement et en résolvant des énigmes.

---

## ▶️ Lancer le jeu

Le jeu fonctionne sur **Windows, macOS et Linux**. Une seule chose est nécessaire :
**Java 17 (ou plus récent)** doit être installé sur l'ordinateur.

> 👉 Java n'est pas installé ? Téléchargez-le gratuitement ici : **https://adoptium.net**
> (choisissez « Temurin 17 » ou plus récent, puis installez normalement.)

Ensuite, **double-cliquez** sur le fichier correspondant à votre système :

| Système        | Fichier à lancer            |
|----------------|-----------------------------|
| 🪟 Windows     | **`Blurred-Windows.bat`**   |
| 🍎 macOS       | **`Blurred-macOS.command`** |
| 🐧 Linux       | **`Blurred-Linux.sh`**      |

> 💡 Astuce : vous pouvez aussi lancer directement **`Blurred.jar`** (double-clic)
> si Java est associé aux fichiers `.jar` sur votre machine.

*Remarque macOS : au premier lancement, si un avertissement de sécurité apparaît,
faites un clic droit sur le fichier → « Ouvrir » → « Ouvrir ».*

*Mac Apple Silicon (puces M1/M2/M3) : le jeu tourne via un **JDK Intel (x64)** —
si vous avez installé Temurin 17 standard, Rosetta gère cela automatiquement au
premier lancement.*

## 🕹️ Contrôles

| Action | Commande |
|--------|----------|
| Interagir (objets, portes, indices) | **Clic gauche** |
| Avancer dans un dialogue | **Clic gauche** |
| Changer de vue dans une pièce | Flèches **◀ ▶** à l'écran |
| Se déplacer entre les pièces | **Mini-carte** en bas à gauche |
| **Ouvrir / fermer le menu PAUSE** | Touche **ÉCHAP** |

Un rappel « **Appuyez sur ÉCHAP pour accéder au menu PAUSE** » est affiché en bas
à droite pendant la partie. Le menu PAUSE permet aussi de changer la **langue**
(Français / Anglais), la luminosité et le son, et d'afficher la **note de version**.

---

## 🐞 Petits soucis d'affichage & solutions

Certains bugs d'affichage viennent du système et se règlent en 2 secondes :

| Problème | Solution |
|----------|----------|
| **Le message "Appuyez sur le bouton ECHAP pour accéder au menu PAUSE" vous empêche de cliquer sur "QUITTER"** | Cliquez sur la partie inférieure du bouton. |
| **Images rétrécies sur fond blanc** (après être sorti d'un zoom) | **Déplacez légèrement la fenêtre** du jeu (clic-glisser sur la barre de titre) pour rafraîchir l'affichage. |
| **Le jeu semble bloqué** et ne réagit plus aux clics | Une fenêtre d'info/avertissement s'est ouverte **derrière** la fenêtre principale : faites **`Alt + Tab`** (ou **`Windows + Tab`**) et sélectionnez-la, puis fermez-la. |

Ces astuces sont aussi rappelées dans la **note de version** du menu PAUSE.
